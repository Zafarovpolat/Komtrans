# КомТранс Азия — HTML/CSS вёрстка

Pixel-perfect статичная вёрстка по&nbsp;Figma-макету. Чистый HTML + CSS, без сборщиков.

## Структура

```
site/
├── index.html         # Главная страница
├── 404.html           # Страница не найдена
├── privacy.html       # Политика приватности
├── css/
│   ├── main.css       # Базовые стили, header, hero, типы, маршруты, футер
│   ├── sections.css   # Преимущества, цифры, калькулятор, шаги, гарантии, портфолио, отзывы, FAQ
│   ├── popups.css     # Модальные окна (call-me, question, calculator, thx, portfolio, review, menu)
│   └── responsive.css # Tablet & mobile (breakpoints: 1280px, 768px)
├── js/
│   └── main.js        # Только модалки и аккордеоны (FAQ, преимущества). По заданию — без сложной логики.
├── fonts/             # Сюда положить Formular-Regular.woff2 и Formular-Medium.woff2
└── assets/
    ├── icons/         # SVG-иконки (logo, иконки типов груза, стрелки и т.п.)
    └── img/           # JPEG/PNG-изображения (hero, фоны секций, портфолио)
```

## Шрифты

В&nbsp;Figma использован шрифт **Formular** (weights 400, 500). Его коммерческая
лицензия лежит вне пакета. Чтобы&nbsp;включить его натив&shy;но,
положите в&nbsp;`/fonts`:

- `Formular-Regular.woff2`
- `Formular-Medium.woff2`

CSS-правила `@font-face` уже&nbsp;подключены и&nbsp;подхватят файлы автоматически.

В&nbsp;качестве визуально-близкого fallback подключён **Inter** через Google Fonts —
поэтому страница корректно отображается «из&nbsp;коробки».

## Запуск

Это статичный сайт, не&nbsp;требующий сборки или сервера. Достаточно открыть
`index.html` в&nbsp;браузере. Для удобства можно запустить лёгкий локальный сервер,
например:

```bash
python3 -m http.server 8000
# или
npx serve .
```

## Адаптивность

- **Desktop:** ≥1281&nbsp;px (макет рассчитан на&nbsp;1920&nbsp;px, контейнер 1344&nbsp;px).
- **Tablet:** 769–1280&nbsp;px (двухколоночные сетки, упрощение шапки).
- **Mobile:** ≤768&nbsp;px (макет 360&nbsp;px, гамбургер-меню, иконка телефона).

## Модальные окна

Все popup-экраны из&nbsp;Figma реализованы как&nbsp;`<div class="modal">` внутри
страницы и&nbsp;открываются по&nbsp;клику на&nbsp;кнопках с&nbsp;`data-modal="..."`:

| Триггер                          | Модалка                       |
|----------------------------------|-------------------------------|
| `data-modal="call"`              | Заказать звонок (call-me)     |
| `data-modal="question"`          | Задать вопрос                 |
| `data-modal="calculator"`        | Калькулятор стоимости         |
| `data-modal="thx"`               | Спасибо за&nbsp;заявку        |
| `data-modal="portfolio"`         | Кейс из&nbsp;портфолио        |
| `data-modal="review"`            | Полный отзыв клиента          |
| `data-modal="menu"`              | Мобильное меню                |

## Замечания по&nbsp;точности

Вёрстка собрана 1:1 с&nbsp;токенами из&nbsp;Figma:

- **Цвета:** `#f94231` (бренд), `#08090a / #181e23 / #1c1c1c` (тёмные), `#f4f2f1`
  (фон), `#7a7c7f` (вторичный текст).
- **Типографика:** размеры от&nbsp;12&nbsp;px до&nbsp;72&nbsp;px, line-height 100% для
  заголовков, letter-spacing -0.04em для крупных заголовков (-0.01em для мелкого
  текста), как в&nbsp;дизайн-токенах Figma.
- **Контейнер:** 1344&nbsp;px по&nbsp;центру, padding 32&nbsp;px.
- **Скошенные углы** карточек (clip-path) воспроизводят треугольный «обрез»
  на&nbsp;CTA-карточках в&nbsp;макете.

Если в&nbsp;браузере шрифт Formular не&nbsp;установлен&nbsp;— подключится Inter,
визуально близкий к&nbsp;Formular (геометрический гротеск). Для абсолютной
1:1-точности достаточно положить файлы Formular в&nbsp;папку&nbsp;`fonts/`.
